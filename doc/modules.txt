Все модули системы должны упаковываться в PHAR архивы.

Архив должен содержать:
- файл класса (название class.php)
- XML файл, содержащий регистрационную информацию
- Файл README, содержащий описание и документацию по методам модуля
Архив может содержать:
- Ресуры, используемые модулем (грфика, шаблоны)
- Другие *.php файлы, которые подключаются к class.php

Все ресурсы должны находиться в соответствующих каталогах:
tpl - шаблоны
img - изображения
js - JavaScript приложения
